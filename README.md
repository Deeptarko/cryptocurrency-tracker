## Installation

Install my-project with npm

```bash
  npm install cryptocurrency-tracker
  cd cryptocurrency-tracker
```
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/Deeptarko/cryptocurrency-tracker.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
